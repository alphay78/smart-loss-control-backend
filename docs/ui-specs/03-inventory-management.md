### Required Fields:

| Field | Type | Validation | Backend Field |
|-------|------|------------|---------------|
| Product | Dropdown | Required | `sku_id` |
| Ordered Qty | Number | Min 0 | `ordered_qty` |
| Received Qty | Number | Min 0 | `received_qty` |
| Cost Price | Currency | Min 0 | `cost_price` |
| Selling Price | Currency | > cost_price | `selling_price` |
| Supplier | Text | Optional | `supplier_name` |

### Backend API:
```javascript
POST /inventory/restock
Authorization: Bearer <owner_token>
{
  "sku_id": "uuid",
  "ordered_qty": 100,
  "received_qty": 98,
  "cost_price": 18500,
  "selling_price": 21000,
  "supplier_name": "Lagos Distributors Ltd"
}

// Response
{
  "success": true,
  "message": "Restock recorded successfully",
  "discrepancy": -2,
  "inventory_after": 98
}
```

### Design Notes:
- Auto-calculate discrepancy (received - ordered)
- Show warning if discrepancy != 0
- Validate selling_price >= cost_price
- Show success toast on completion
- Auto-navigate back to inventory list
